Web网页开发

什么是HTML

HTML（HyperText Markup Language）是一种超文本“标记”语言，并非编程语言。

HTML的起源

HTML起初是为了能够将文字和图片关联起来，用一台发送或接收。

标记到底是什么样的

标记通常就是我们所说的标签，它是由小于符号和大于符号包裹一个关键词而组成的，通常是成对出现的、

现在市场上主流五大浏览器

每种浏览器都有自己的内核(引擎)（解析网页的一个程序，io以什么方式去渲染它都要通过引擎去执行）
目前主流的浏览器分为五种

  • Chrome谷歌浏览器 （Webkit内核，V8 js引擎）

  • Firefox火狐浏览器 （Gecko内核）

  • Internet Explorer IE浏览器（ Trident内核）

  • Opera 欧朋浏览器 （Presto内核） 主要市场：移动端

  • Safari 苹果浏览器 （Webkit内核，但JS引擎为Nitro）

    不需要去记，简单了解下

网页文件扩展名

.html .htm 这两种是比较常见的，在早期系统中文件名是有8+3组成 三个扩展名所以不支持四个字母的扩展采用.htm 现在通常使用.html作为扩展名

HTML标签使用规范

  1. 有闭合标签的必须闭合 例如：
      …
  2. 所有标签名一律小写
  3. 代码缩进，使阅读代码更加易懂
  4. 命名规范，见名之意

编辑器 Description

配置模板

Tag Description
!DOCTYPE HTML HTML5的文档类型声明标准，写在第一行，不是标签，主要用于文档类型的声明
charset=”utf-8” 设置文档的字符编码集，供浏览器解析

简单了解下，不需要掌握

<!doctype html> <!-- 网页文档声明 -->
<html>
    <head>
      <meta charset="utf-8"><!-- 设置网页字符编码集 -->
      <meta name="keywords" content=""><!-- SEO关键字 -->
      <meta name="description" content=""><!-- SEO网页八十字以内描述信息 -->
      <title>网页标题</title><!-- 网页标题 --> 
    </head>
      <body><!-- 标签写在body里面 -->

    </body>
</html>

HTML树状结构

H标签

h1~h6随着数字增大，字体越小，始终处于加粗的状态，用于强调文本
前后的元素都被换行了

H1在一个页面当中只能写一次，利于SEO，搜索引擎会爬取这个标签中的关键词

P段落标签

<p>
    纤云弄巧，飞星传恨，银汉迢迢暗度。金风玉露一相逢，便胜却、人间无数。柔情似水，佳期如梦，忍顾鹊桥归路。两情若是久长时，又岂在、朝朝暮暮。
</p>

前后的元素都会被换行

锚点文本A超链接标签

<a href="#">跳转</a>

href属性必须要写，如果不 写则会显示普通文本一样没有任何效果
# 表示当前页面，href=”#”也就是跳转到当前页面了
javascript: void(0); 执行js表达式0，并且没有返回值，所以不会执行任何操作，而#默认表示Top位置

锚点标记

<a href="#flag"></a>

hr水平分割线

一条单独在一行的水平线

学习之前最基本的图片格式需要简单了解下

图片格式

图片格式 描述
BMP 占用空间大，色彩丰富
JPEG 压缩方式通常是破坏数据性压缩，在压缩过程中图像的质量会遭受到课件的破坏
GIF 对 透明色 和 多帧动画 的支持
PNG 无损压缩的位图 格式，支持Alpha通道的 透明/半透明 特性

图片标签

<img src="图片路径" alt="仅供搜索引擎识别图片" width="图片宽" height="图片高" />
属性 描述
src 相对路径、绝对路径、http网络图片、当前目录跳上层../
alt 用来描述图片内容，仅供HTML识别该图片内容，另方面供搜索引擎识别图片
width 按照此宽度显示、当失去比例则会拉伸
height 按照此高度显示、当失去比例则会拉伸
title 鼠标在图片上时候，显示文字信息

width/height宽高强调根据图片宽高来给定，如果没有宽高则会去计算图片的宽高度，计算的时候会浪费一些时间，虽然不是很明显

相对路径

相对于当前文件下的目录

绝对路径

指定了盘符（驱动器）

上层目录

../../../../../../../../ 跳转到相对路径的上层目录

无序列表

<ul>
    <li>111</li>
      <li>222</li>
      <li>333</li>
</ul>

| 属性 | 描述 |
| disc | 默认值，小黑圆点 |
| square | 实心方块 |
| circle | 空心圆 |

PS：默认无序列表ul会有小黑圆点，部分浏览器版本可能呈现空心圆点
使用范围，一些比较有规律的一行或是一列列的通常都会采用无序列表使用相当多，比如导航条、文章列表、帖子列表等

有序列表

<ol>
    <li>111</li>

</ol>
属性 描述

a 小写字母排序
A 大写字母序列号
i 罗马小写字母排序
I 罗马大写字母排序

表格

<table>
    <tr>
          <td>111</td>
        <td>222</td>
    </tr>

属性 描述

td table data 代表列

caption 表格标题添加

        </tr>

</table>
